# Competitor Benchmark Audit

## Competitor categories

### Stat tracking apps

Usually strong at scorekeeping and weak at athlete-facing beauty/social distribution.

### School/team websites

Often public but slow, inconsistent, ugly, and hard to update.

### Recruiting/profile platforms

Often serious and useful but may feel static or paywalled.

### Social platforms

Great for media and attention but lack structured stats, verification, and game context.

### Sports data providers

Credible data but not designed around individual high school/college athlete expression.

## Statman opportunity

Blend:

- Athlete profile aesthetics.
- Game-day stat capture.
- Social feed energy.
- Structured sports data.
- Verification/dispute trust.
- Sport-specific input UX.

## Benchmark dimensions

- Mobile profile beauty.
- Speed of stat entry.
- Live game support.
- Public shareability.
- SEO profile pages.
- Data credibility.
- Media integration.
- Sport flexibility.
- Team manager workflow.
- Fan discovery.

## Differentiation statement

Statman should not win by being the deepest box-score software first. It should win by making athlete profiles look better and making game updates instantly become profile content.
