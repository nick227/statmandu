# Asset Pack / Media Kit

## MVP asset needs

### Brand assets

- Statman logo.
- App icon.
- Favicon.
- Social preview card background.

### Demo sports assets

- One demo league logo.
- Two demo team logos.
- Twenty player placeholder/action images.
- Team color swatches.
- Basketball court/texture fallback image.

### UI assets

- Sport icons.
- Verification icons.
- Dispute/warning icon.
- Live indicator icon.
- YouTube icon.
- Share card templates.

### Media requirements

- Use YouTube embeds for MVP video.
- Use responsive images with multiple sizes.
- Provide fallback media for players, teams, games, leagues.

## SmartImage behavior requirements

- Correct size selection.
- Lazy loading.
- Aspect ratio lock.
- Placeholder state.
- Error fallback.
- Optional dominant color fallback.
- Alt text support.
- Loading indicator for slow media.
