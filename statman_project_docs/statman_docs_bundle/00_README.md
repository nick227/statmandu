# Statman Pre-Project Documentation Bundle

This bundle contains the first 25 useful kickoff documents for Statman: a sport-agnostic, mobile-first public athlete profile platform with live game stat capture, media-first entity pages, social feed, verification/disputes, source imports, and OpenAPI-driven architecture.

## Current product stance

- Primary audience: athletes.
- Secondary audience: sports lovers/fans.
- Core product: public athlete profiles.
- First demo: basketball, one league, two teams, twenty players.
- Platform: React Native Web/PWA first.
- Data: manual entry, imports, public seeded profiles, source tracking, verification, disputes.
- Media: YouTube-first MVP.
- Architecture: OpenAPI-first REST contracts, separate realtime event contracts, generated SDK, sport configuration, event-sourced live game capture.

## How to use this bundle

Start with these three docs first:

1. `01_PRD.md`
2. `05_SITE_ARCHITECTURE_MAP.md`
3. `12_API_DOCUMENTATION.md`

Then use the UX, data, and realtime docs to guide design and implementation.
