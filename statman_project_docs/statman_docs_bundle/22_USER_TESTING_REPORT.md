# User Testing Report Template

## Testing goals

- Validate athlete-first profile appeal.
- Validate live stat capture speed.
- Validate homepage content model.
- Validate Explore/Home separation.
- Validate trust labels and dispute footnotes.

## Prototype tasks

### Athlete

1. Create or claim a profile.
2. Add a YouTube highlight.
3. Review profile preview.
4. Share a stat card.

### Scorekeeper

1. Start a basketball game.
2. Record five possessions.
3. Undo a mistake.
4. End game and resolve a conflict.

### Fan

1. Find a live game.
2. Follow a player.
3. React to a big game card.
4. Open Explore and find top scorers.

## Questions to answer

- Does the profile feel impressive enough for an athlete to share?
- Can stat entry happen without typing?
- Are source and dispute labels clear without adding clutter?
- Does the homepage feel alive?
- Does Explore feel separate and more serious?
- Are gestures intuitive or confusing?
