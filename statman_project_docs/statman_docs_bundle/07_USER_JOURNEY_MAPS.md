# User Journey Maps

## Journey 1: Athlete creates a profile

1. Lands on homepage.
2. Searches for existing profile or starts new profile.
3. Adds name, sport, position, team, class year.
4. Adds YouTube highlight link.
5. Adds initial stats manually or imports from team/game.
6. Sees generated profile preview.
7. Shares profile or stat card.
8. Gets follows/reactions.

## Journey 2: Team manager captures a live game

1. Opens Enter.
2. Selects team and game.
3. Joins as team scorer.
4. Uses basketball event pad.
5. Captures events offline/online.
6. Reviews recent events and undoes mistakes.
7. Ends game.
8. Reviews conflicts against other reporters.
9. Finalizes box score.
10. Athlete profiles update.
11. Share cards are generated.

## Journey 3: Fan follows a live game

1. Opens homepage or team profile.
2. Taps live game.
3. Enters spectator mode.
4. Watches scoreboard, top performers, and timeline.
5. Reacts to moments.
6. Follows a player.
7. Shares game or player profile.

## Journey 4: User disputes a stat

1. Views player profile or game box score.
2. Taps disputed/incorrect stat action.
3. Submits corrected value and optional evidence/source.
4. Stat gets marked in review.
5. Team manager/admin sees dispute.
6. Dispute is resolved or published with footnote.

## Journey 5: Seeded profile is claimed

1. Athlete finds imported public profile.
2. Taps Claim Profile.
3. Completes verification step.
4. Adds media and bio.
5. Confirms or disputes existing stats.
6. Profile becomes richer and more trustworthy.
